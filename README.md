# E-Commerce Website

This project is a simple E-Commerce Website frontend developed using HTML, CSS, and JavaScript.  
It demonstrates the basic structure and functionality of an online shopping website.

## Project Features

- Homepage with product listings
- Product prices and ratings
- Add to Cart functionality
- Cart item count and total price calculation
- Login page for users
- Responsive and simple user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git and GitHub

## Project Structure

├── homepage.html

## How to Run the Project

1. Clone the repository:
   git clone https://github.com/USERNAME/e-commerce-website.git

2. Open the project folder.

3. Open the file `homepage.html` in a web browser.
